# Playwright Test (/test)

Run comprehensive web application tests using 5 specialized agents.

## Quick Usage

- `/test` - Run all 5 agents
- `/test [url]` - Test specific URL

## Execution

Follow the workflow defined in the **playwright-test** skill:

1. Set up test environment
2. Run Explorer agent (page discovery)
3. Run 4 agents in parallel (Functional, Visual, API, Performance)
4. Generate unified report

Refer to `skills/playwright-test/SKILL.md` for detailed execution steps.
