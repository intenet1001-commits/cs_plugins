# Report Writer v2 (/report)

Create professional Korean business reports with structured formatting.

## Quick Usage

- `/report` - Interactive report creation
- `/report [topic]` - Create report on specific topic

## What's New in v2

- 7-level hierarchy (added o bullet and * note)
- Compact spacing (no excessive gaps)
- Based on 보고서_템플릿_가이드_v4.md

## Execution

Follow the workflow defined in the **report-writer** skill:

1. Gather report requirements
2. Create document structure with 7-level hierarchy
3. Apply Korean typography (나눔명조)
4. Generate .docx file

Refer to `skills/report-writer/SKILL.md` for detailed formatting guidelines.
