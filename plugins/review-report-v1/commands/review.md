# Review Report (/review)

Create professional Korean review and analysis reports with checkbox sections and structured formatting.

## Quick Usage

- `/review` - Interactive review report creation
- `/review [topic]` - Create review report on specific topic

## Document Style

- Checkbox sections (□)
- Hyphen bullets (-)
- Circled numbers (①②③)
- Reference boxes with borders
- Tables with headers

## Execution

Follow the workflow defined in the **review-report** skill:

1. Gather report requirements
2. Create document structure with checkbox sections
3. Apply Korean typography (나눔명조)
4. Generate .docx file

Refer to `skills/review-report/SKILL.md` for detailed formatting guidelines.
